Test and Train DecisionTreeRegresson Model

This Decision Tree Regression model raw data comes from Zillow’s Home Value Index(all homes-mid-tier). Before the prediction models can begin the raw data requires removal of all the zeros and empty data cells. The data then has all date associated columns merged into one column titled ‘year’ and the corresponding values for those dates assigned to a ‘values’ column. I will start predictions using all the data and progressively increase the predictions’ focus until we achieve the optimal accuracy. 33% of the data is used to evaluate the prediction model’s accuracy.

  • The best prediction for Raleigh, North Carolina is 92.98% accuracy with this model. I am certain we could improve that prediction by refining our focus even further.

  • Assign features variable with columns of interest

  • Assign a target variable that we are going to predict for.
