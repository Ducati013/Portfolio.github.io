Begin the building the modeling and testing

  • Assign features variable with columns of interest

  • Assign a target variable that we are going to predict for.

  • I will split the data with a 67% to train and 33% to testing.

  • This is all done in such a way that it is modular

    • I can easily modify this method and test in another configuration to find the most successful prediction model.

features=['SizeRank','year']
x=NC.loc[:,features]
target=['mean_prices']
y=NC.loc[:,target] # target

X_train, X_test, y_train, y_test = train_test_split(x, y, random_state=0, train_size = .67)

reg = DecisionTreeRegressor(max_depth = 2, random_state = 0)
