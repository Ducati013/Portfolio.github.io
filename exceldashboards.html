Excel Dashboard

The data from this dashboard is available on the Yahoo Finance so every can check the data. This is a busy dashboard that shows what sort of things I can create in excel. While this dashboard may be a bit overwhelming it is just meant to be an exhibition of possibilities.

This next dynamic dashboard is equipped with milestone labeled macro buttons and an adjustable time bar. The time bar enables operators to visualize graphed elements’ correlations during that observation period. All the elements in the dashboard are dynamic and respond to when the timeline is changed by the user or when the preset milestones are selected.
